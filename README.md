### AMETHYST MINER
Amethyst miner is a simple  rng based game in which you have a small chance of dropping rare items (gems and artifacts).

This game was created for self-education and learning purposes <u>ONLY</u>, **NOT** for commercial use.

You can play the game from the GitHub pages link here: https://robert0106l.github.io/

There is no backend, just HTML, CSS and JS.

Only **italian** language available.
