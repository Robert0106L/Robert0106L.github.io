### AMETHYST MINER
Amethyst miner is a simple NON-COMMERCIAL rng based game in which you have a small chance of dropping rare items (gems and artifacts).

You can play the game from the GitHub pages link here: https://robert0106l.github.io/

There is no backend, just HTML, CSS and JS.
